Cascade Style Sheet

Assim como no HTML, no CSS também precisamos nos familiarizar com alguns termos decorrentes. Os principais são seletores, propriedades e valores.

↑
↑

Seletores

↑
↑

Conforme vamos adicionando elementos ao HTML, eles estão passíveis de estilização e para conseguirmos aplicar os estilos a esses elementos precisamos indicá-lo ao CSS. Seletores são, na maioria das vezes, as tags HTML como <p>, <h1>, <div> e podem indicar, também, um atributo do elemento, como class e id.

p { ... }

Após os seletores, abrimos e fechamos chaves. O que está entre as chaves são as propriedades e seus valores. No exemplo acima, estamos estilizando os parágrafos da página.

↑
↑

Propriedades

↑
↑

Uma vez selecionado, cada elemento pode ter suas propriedades estilizadas. As propriedades ficam entre as chaves e são seguidas de dois-pontos. Existem inúmeras propriedades que podemos usar e mais propriedades continuam sendo criadas.

p {
color: ...;
font-size: ...;
}

Propriedades começam com dois-pontos e terminam com ponto-e-vírgula. No exemplo acima, estamos estilizando a cor e tamanho da fonte.

↑
↑

Valores

↑
↑

Cada propriedade CSS pode receber um tipo de valor que determina seu comportamento.

p {
color: blue;
font-size: 18px;
}

Os valores vão determinar que os parágrafos vão ter a cor azul e tamanhode 18px.

↑
↑

Class e id

↑
↑

Além de tags HTML, os atributos de classe e id também podem ser usados como seletores mais específicos.

#id {
width: 900px;
display: block;
}

.class {
color: yellow;
font-size: 14px;
}

IDs são indicados usando uma cerquilha (ou hashtag) antes do nome e classes são indicadas com um ponto.
